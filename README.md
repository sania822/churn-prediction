# Customer Churn Prediction

## Problem
Predict whether a telecom customer will churn.

## Model
RandomForestClassifier (tuned using GridSearchCV)

## Accuracy
96% accuracy
Recall (Churn class): 73%

## Tech Stack
- Python
- Scikit-learn
- Flask
- HTML/CSS

## How to Run
pip install -r requirements.txt
python app.py
